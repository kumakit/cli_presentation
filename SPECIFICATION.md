# ターミナル・プレゼンテーション・ツール 仕様書

## 1. 概要
本ツールは、外部のAIエージェントや専用ソフトウェアを利用できない制限された社内環境（Windows / PowerShell）において、コマンドラインインターフェース（CLI）上でのプレゼンテーションを実現するための軽量スクリプトです。
GitHub Copilot CLI Presentation の「AIが喋っているようなタイピング演出」や「キーボードによるスライド遷移」をモデルにしています。

## 2. システム要件
- **OS**: Windows 10 / 11
- **実行環境**: PowerShell 5.1 以上（PowerShell 7 推奨）
- **依存関係**: なし（PowerShell標準機能のみで動作）

## 3. ファイル構成
- `Presentation.ps1`: プレゼンテーション制御ロジック（本体）
- `slides.md`: スライド原稿データ（Markdown形式）

## 4. 機能仕様

### 4.1 スライド解析
- Markdown形式のファイルを読み込み、行頭の `---`（ハイフン3つ）を区切り文字としてスライドを分割します。
- 簡易的なMarkdownパース機能を備えています。
    - `# `: 第1見出し（水色・太字）
    - `## `: 第2見出し（黄色）
    - `- `: 箇条書きリスト（灰色）
    - その他: 通常テキスト（標準色）

### 4.2 表示演出（タイピングエフェクト）
- スライド表示時、テキストを1文字ずつ時間差（デフォルト 20ms）で出力します。
- **演出スキップ機能**: タイピング中に任意のキーを押下すると、そのスライドの残りのテキストを即座に表示します。

### 4.3 ナビゲーション（操作方法）
ユーザーのキー入力を待機し、以下の操作を受け付けます。

| 操作キー | アクション |
| :--- | :--- |
| **右矢印 / Space / Enter** | 次のスライドへ進む |
| **左矢印 / Backspace** | 前のスライドへ戻る |
| **Q / Escape** | プレゼンテーションを終了しターミナルをクリアする |

### 4.4 パラメータ
スクリプト実行時に以下の引数を指定可能です。

- `-FilePath`: 読み込むスライドファイルのパス（デフォルト: `slides.md`）
- `-TypingSpeed`: タイピングの速さ。ミリ秒単位（デフォルト: `20`）

## 5. 実行方法
PowerShellを開き、スクリプトがあるディレクトリで以下のコマンドを実行します。

```powershell
powershell -ExecutionPolicy Bypass -File .\Presentation.ps1
```

## 6. カスタマイズ
スライドの内容を変更するには、`slides.md` をテキストエディタで編集してください。標準的なMarkdown記法で記述でき、独自のASCIIアートなどを埋め込むことも可能です。
