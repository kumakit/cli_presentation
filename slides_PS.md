# PowerShellの魅力 ─ 直感的で美しく書ける！

## オブジェクト指向とわかりやすさ

- コマンド出力がオブジェクト
    → テキストではなく「構造化データ」として扱える 😭

- 動詞-名詞の直感的なコマンド名

    ┌─────────────────────────────────────┐
    │  Get-Service   … サービスを取得     │
    │  Start-Service … サービスを起動     │
    │  Stop-Service  … サービスを停止     │
    └─────────────────────────────────────┘

## パイプラインで処理をつなぐ

- 小さなコマンドをパイプ "|" で連結してデータを流す

    サービス取得 → 起動のみフィルタ → ソート → 出力

    ★ デモ：ワンライナーで起動中サービス一覧を取得！
